# Gestore_Film_IngSW
Repository dedicata al progetto "Gestore di una collezione di film" nel contesto del corso di Ingegneria del Software A.A 2024-2025.

# Descrizione
Applicazione desktop in Java Swing per la gestione di una collezione personale di film. 
Offre le funzioni base di inserimento, modifica e rimozione di film. Inoltre, sono supportate le funzioni di ricerca, filtro e ordinamento 
dei volumi contenuti all’interno della collezione. 
Offre anche la possibilità di salvare e caricare da file JSON o CSV e supporto a operazioni di tipo Undo/Redo.
